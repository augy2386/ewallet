# ewallet
Sistem aplikasi topup dan redeem saldo member
sistem ini dapat diakses jika sudah mendaftar jadi member dan member yang sudah terdaftar
modul di sistem ini :
- modul pendaftaran member baru
- modul login member (menggunakan email dan password yang terdaftar saat pendaftaran)
- modul top up saldo
- modul redeem / pengurangan saldo
- modul history transaksi (topup/redeem)




Cara pakai singkat:
1. Ekstrak ke htdocs, misal folder: htdocs/ewallet
2. Import SQL dari folder 'sql/ewallet_db.sql' ke MySQL.
3. jalankan http://localhost/ewallet/

Teknologi : 
- Codeigniter 3
- Database : MySQL
